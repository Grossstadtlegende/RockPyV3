Fabian2003¶

Plots.hysteresis.plot_hys(hys_obj, ax, norm_factor=1, out='show', folder=None, name='output.pdf')[source]¶

Main plotting of a hysteresis. :param hys_obj: hysteresis object

Parameters:
  • ax – pyplot.ax object
  • norm_factor – normalization of the y-data
  • out –

    choice of:

    • ‘show’ : shows the plot
    • ‘rtn’ : returns the plot
    • ‘save’ : saves a pdf of the plot. a folder has to be specified. Name can be specified
  • folder – str where the pdf should be saved
  • name – str name of the pdf
